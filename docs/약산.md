# 약산 ↔ 금일 / 약산 ↔ 생일 방면 (완도군 섬↔섬 차도선)

> 완도군 내 섬↔섬 여객·차도선 노선. 완도 본섬 터미널을 경유하지 않는 별도 노선이라
> 완도 출발/도착 탭과 구분해 표시한다(포항 KTX 블록과 유사한 별도 섹션).
> **결론: 두 노선 모두 MTIS 운항 스케줄 API에 실시간 등록됨 → 정적 데이터 없이 실시간 구현 가능.**

---

## 노선 개요

| 표시 노선 | 실제 항구 | MTIS 출발항/도착항 (`oport_nm`/`dest_nm`) |
|---|---|---|
| 약산 ↔ 금일 | 약산도 **당목항** ↔ 금일도 **일정항** | `당목` ⇔ `일정` |
| 약산 ↔ 생일 | 약산도 **당목항** ↔ 생일도 **서성항** | `당목` ⇔ `서성` |

- **약산도**: 완도군 약산면. 본섬(고금도)과 약산연도교로 연결돼 차량 진입 가능 → 당목항이 인근 섬 환승 거점.
- 운영: **완도농협 약산지점**(완농페리 선단) 외 일부 편 풍진메이슨·평화페리9호 공동 운항.

### 운영 선박 (MTIS `psnshp_nm`)

| 노선 | 선박 |
|---|---|
| 약산 ↔ 금일 (당목⇔일정) | 완농페리3호 · 풍진메이슨 · 평화페리9호 |
| 약산 ↔ 생일 (당목⇔서성) | 완농페리5호 · 완농페리호 |

---

## MTIS 실시간 연동 정보

- **API**: 기존 `getMtisDay`(MTIS 운항 스케줄) 그대로 사용 — 별도 호출 불필요, 전국 데이터에 이미 포함됨.
- **groupKey 매핑** (구현 시 `ferry.ts`에 추가할 분기):
  - 약산↔금일: `oport/dest`가 `당목`↔`일정`, `nvg_seawy_nm` = `"당목일정"` / `"일정당목"`
  - 약산↔생일: `oport/dest`가 `당목`↔`서성`, `nvg_seawy_nm` = `"당목서성"` (일부 `"당목서성-화전-서성"` = 화전 경유)
- **중복 병합 주의**: 당목→서성에 18:00/18:01처럼 5분 이내 중복 편 존재 → 기존 `deduplicateTimes`로 자동 병합됨(8편→7편).
- ⚠️ **당목↔녹동**(약산↔녹동, 고흥 방면) 편도 MTIS에 있으나 이번 대상 아님 — `녹동` 키워드는 제외할 것.
