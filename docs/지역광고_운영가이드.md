# FerryCast 지역 광고 운영 가이드

> 완도 메인(`/`) 하단 지역 광고 슬롯의 영업·운영 기준.
> 오퍼(문의 메일)가 오면 이 문서대로 대응한다.

---

## 1. 현황

- **적용 지역**: 완도·포항·목포·인천 **전 지역** (카카오 애드핏 전면 대체)
- **자리**: 각 지역 메인 첫 화면 하단 (쿠팡 특산물 섹션 아래) — `src/components/LocalAdSlot.tsx` 공용
- **광고 안내 페이지**: 지역별 시안 미리보기 (`AdsPageContent.tsx` 공용)
  - 완도 `https://ferrycast.kr/ads` · 포항 `/pohang/ads` · 목포 `/mokpo/ads` · 인천 `/incheon/ads`
  - 소개·게재위치는 지역명 표기, **게재 예시(시안)는 지역 무관 공용 문구**
  - 메일 문의 정보는 `src/lib/adInquiry.ts`의 `buildAdMailto({regionName, adsPath})` — 제목·본문·미리보기 링크가 지역별로 자동 생성
- **트래픽**: 완도 일 방문 100여 명 (2026년 7월 기준). 포항·목포·인천은 별도 집계 필요 — **실측 전까지 지역별 방문자 수를 영업 문구에 넣지 않는다**(허수 금지)
- **방문자 특성**: 상당수가 여행·이동을 준비하는 방문자 (완도는 블로그 유입 중심)
- **강점**: ① 여행 직전 타겟 ② 배 시간 확인차 하루 여러 번 재방문 → 반복 노출
- **문의 채널**: 메일만 (전화번호 비공개)
- **복수 광고주 방침**: 광고주가 늘면 **10초 간격 자동 교체(로테이션) 배너**로 운영 — "단독 게재"는 영업 문구에서 사용하지 않는다

## 2. 금액 정책

| 상품 | 노출 위치 | 금액 | 조건 |
|---|---|---|---|
| 기본 | 완도 메인 하단 | **월 5만원** (정가) | 월 단위 선불, 계좌이체 |
| **타겟 패키지** | 메인 + **해당 항로 상세 화면 하단** | **월 5~7만원** | 업체와 항로가 맞을 때만 (예: 청산도 펜션 → 청산도 상세) |
| 첫 광고주 프로모션 | 기본 자리 | **첫 달 무료 → 이후 월 3만원** | 실적 사례 확보가 목적 |
| 3개월 묶음 | 기본 자리 | 월 3만원 (총 9만원) | 최대 3개월 단위까지만 |

**타겟 패키지 영업 포인트**
- "청산도 가는 손님에게만 보여드립니다" — 상세 화면은 해당 항로를 실제로 이용하는 사람만 열어보는 화면이라, 메인보다 매칭 정확도가 높다.
- 상세 화면은 시간표·요금·예약까지 확인하는 화면이라 체류 시간이 길어 시선이 오래 머문다.
- 첫 광고주 계약 시 업셀 옵션으로 제안. **상세 화면 배너는 계약 확정 후 제작** — 빈 모집 슬롯을 상세 화면에 미리 깔지 않는다(모집 문구 반복 노출은 "안 팔리는 앱" 인상을 줘 영업에 역효과).

**원칙**
- 6개월~1년 장기 선불 계약 금지 — 트래픽 성장 시 가격 조정 여지를 남긴다.
- 월 1만원만 받아도 애드핏 수익(월 수천 원)보다 이득 — 가격 협상 여지 있음. 단 무료 연장은 금지(첫 달 무료 1회만).
- 비교 논리: 지역 현수막 10~20만원/월, 지역신문·전단지 10~30만원(1회성) 대비 저렴 + 타겟 정확.

## 3. 오퍼 대응 절차

1. **24시간 내 회신** — 소상공인은 답이 늦으면 식는다.
2. 회신에 포함: 게재 위치 스크린샷 + Vercel Analytics 방문자 캡처 1장 + 금액·기간 + "배너 무료 제작"
3. **배너 제작 포함이 성사의 핵심** — "가게 사진 2~3장 + 소개 한 줄만 주세요"
4. 입금 확인 → 게재 → 게재 화면 캡처 전송
5. **월말 간단 리포트** (노출 일수·클릭 수) — 갱신율을 좌우한다. GA4 클릭 이벤트 기반.

## 5. 배너 시안 (3종)

> 광고주 확정 시 아래 중 업종에 맞는 형태로 제작. 공통 규격은 6번 참고.

### 시안 A — 텍스트형 (기본, 모든 업종)

```
┌──────────────────────────────────────────┐
│ 광고                                      │
│ 🏠 청산도 앞 ○○펜션                        │
│ 선착장에서 5분 · 전 객실 오션뷰             │
│ 예약 061-000-0000        자세히 보기 →     │
└──────────────────────────────────────────┘
```
- 업체명(굵게) + 한 줄 소개 + 전화번호 + 링크 화살표
- 슬롯과 동일한 카드 스타일(rounded-2xl), 배경 흰색, 좌측 업종 이모지

### 시안 B — 사진 카드형 (펜션·식당 등 비주얼 업종)

```
┌──────────────────────────────────────────┐
│ 광고                                      │
│ ┌────────┐  ○○횟집 — 완도 활전복 전문      │
│ │ 가게   │  터미널 도보 3분 · 포장 가능     │
│ │ 사진   │  ☎ 061-000-0000               │
│ └────────┘             메뉴 보기 →        │
└──────────────────────────────────────────┘
```
- 좌측 정사각 썸네일(가게·음식 사진) + 우측 텍스트 2줄 + 전화
- 사진은 광고주 제공 → 우리가 크롭·보정

### 시안 C — 혜택 강조형 (프로모션 있는 업체)

```
┌──────────────────────────────────────────┐
│ 광고                                      │
│ 🎁 "페리캐스트 보고 왔어요" 하면            │
│    ○○카페 아메리카노 500원 할인            │
│    완도터미널 맞은편 · 07:00 첫 배부터 영업  │
└──────────────────────────────────────────┘
```
- "페리캐스트 보고 왔다"는 트리거 문구 → **광고 효과가 오프라인에서 측정됨**
- 광고주가 효과를 체감하기 가장 좋은 형태라 갱신율에 유리

### 배너 카피 작성 원칙
- 첫 줄에 **위치 정보**(터미널에서 몇 분)를 반드시 — 방문자는 지금 배 타러 가는 사람
- 전화번호는 탭하면 바로 걸리게 `tel:` 링크
- 과장 문구(최고, 유일 등) 금지 — 서비스 신뢰도와 직결

## 6. 배너 공통 규격 (개발용)

| 항목 | 값 |
|---|---|
| 위치 | `LocalAdSlot` 자리 교체 (완도 메인 하단) |
| 형태 | 앱 카드 스타일 통일 — rounded-2xl, border, 흰 배경 |
| 라벨 | 좌상단 `광고` 라벨 **필수** (기존 애드핏 표기 원칙과 동일) |
| 링크 | 광고주 사이트/네이버플레이스/전화(`tel:`) 중 택1 |
| 추적 | GA4 클릭 이벤트 (`local_ad_click`) 심기 — 월간 리포트 근거 |
| 복수 게재 | 광고주 2곳 이상이면 **10초 간격 자동 교체 캐러셀**로 전환 (쿠팡 특산물 배너와 동일 방식) |
| 타겟 패키지 | 해당 항로 `RouteDetail` 맨 하단(해운조합 예약 버튼 아래)에 게재 — 핵심 정보(시간표·요금·예약)를 밀지 않는 위치. 계약 확정 후 제작, 모집 슬롯은 상세 화면에 두지 않음 |
| 파일 | 광고주 확정 시 `LocalAdBanner.tsx` 신규 제작, 미계약 기간엔 `LocalAdSlot`(모집 문구)으로 복귀 |

## 7. 세금 유의

- 직거래 광고비는 애드핏·쿠팡과 달리 **직접 신고 대상**
- 소액일 땐 기타소득 처리 가능, 광고주가 늘면 **사업자등록(간이과세) 검토**
- 사업자 없으면 세금계산서 발행 불가 → 광고주에게 "계좌이체, 세금계산서 불가" 사전 고지
